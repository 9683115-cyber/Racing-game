# 🏁 Simple Racing Game

A fun, simple 2D top-down racing game built with HTML, CSS, and JavaScript!

## Features

- **2 Tracks**: City and Desert
- **2D Top-Down View**: Classic arcade racing style
- **Smooth Physics**: Acceleration, friction, and realistic car rotation
- **Live HUD**: Speed, track name, and progress tracking
- **Simple Controls**: Easy to learn and play

## How to Play

1. Open `index.html` in any web browser
2. Select a track (City or Desert)
3. Use arrow keys or WASD to move
   - **Arrow Up / W**: Accelerate forward
   - **Arrow Down / S**: Reverse
   - **Arrow Left / A**: Turn left
   - **Arrow Right / D**: Turn right
   - **Spacebar**: Brake
4. Race to the finish line at the top of the track!

## Game Controls

| Key | Action |
|-----|--------|
| ↑ / W | Forward |
| ↓ / S | Reverse |
| ← / A | Turn Left |
| → / D | Turn Right |
| Space | Brake |

## How It Works

- **Car Physics**: The car accelerates smoothly and has friction applied each frame
- **Track Design**: Each track has obstacles to navigate around
- **Progress Tracking**: HUD displays your current speed and progress to the finish line
- **Win Condition**: Reach the finish line (top of the track) to complete the race
